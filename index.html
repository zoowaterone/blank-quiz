⚙️ 설정
☁️ GitHub Gist 동기화
GitHub Personal Access Token을 입력하면 모든 기기에서 같은 데이터를 사용할 수 있어요.
토큰 발급: github.com/settings/tokens → gist 권한만 체크하면 됩니다.
✅ GitHub 토큰이 연결되어 있습니다.
💡 사용 방법
1. 토큰 연결 후 업로드로 현재 기기 데이터를 저장하세요.
2. 다른 기기에서 같은 토큰으로 연결 후 다운로드하면 동기화됩니다.
🗄️ 데이터 관리
로컬 데이터를 JSON 파일로 백업하거나 복원할 수 있어요.
